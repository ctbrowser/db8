==================
||Tv testing 0.1||
==================

================
||Requirements||
================

* node.js -- v0.8.7 or newer
* ssh2 -- an ssh2 client module for node.js
* keypress npm for interaction with user through keyboard

=================================
|| Installing required modules ||
=================================
* ssh2 -- "npm install ssh2"
* keypress -- "npm install keypress"

================
|| How to run ||
================
* copy example_config.json file to config.json and write an IP-adress and password of TV.
* execute "$ node jsTvTest.js"
